## Get Started(`CI/CD`)

### Docker Image 만들기

```shell
* docker image rm $(docker image ls -f=dangling=true -q)
* docker build -t jobmp-front-docker-image:0.0.1-SNAPSHOT .
* docker run jobmp-front-docker-image:0.0.1-SNAPSHOT
* docker-compose up -d
```

### Nginx UP/Down 

```shell
sudo nginx -t
sudo service nginx restart
```
